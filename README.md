## 프로젝트 소개
#### 배경 및 필요성
- 사과는 전체 과수작물 중 생산량 1위
- 농업 종사자 근골격계 질환 발생율 증가 추세
- 선별 공정에서의 시간과 비용 소모 多

- 이미지 분석
- 높이조절 리프트
